- Tailwind + TypeScript

## Run the Code!

```bash

# or
yarn
# or
pnpm install
```

Then to run
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Visit http://localhost:3000 in your browser to view the gesture detection in action.
